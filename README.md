# MyGitProject

COMP1243 Assignment 1

## Goal

The goal of this assignment is to make a simple terminal based calculator. This will input two numbers and perform a calculation on them.

## Roles

- Shahbaz: Assigning roles and breaking down the problem.
- Manav: Writing the code and reviewing the code.
- Mantaj: Support, reviewing the code and testing the code.

## Problem Breakdown

- Write a function that takes two numbers as input. - Shahbaz
- Write 4 four functions that perform the four basic operations (addition, subtraction, multiplication, division). - Manav
- Then show a menu of operations to choose from. - Shahbaz
- Based on the choice, call the appropriate function and display the result. - Mantaj
