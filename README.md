# MyGitProject

COMP1243 Assignment 1

## Goal

The goal of this assignment is to make a simple terminal based calculator. This will input two numbers and perform a calculation on them.

## Problem Breakdown

- Write a function that takes two numbers as input.
- Write 4 four functions that perform the four basic operations (addition, subtraction, multiplication, division).
- Then show a menu of operations to choose from.
- Based on the choice, call the appropriate function and display the result.
